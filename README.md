# Domain Adaptive NLP System for Sentiment Analysis

## 📌 Description
Domain-adaptive sentiment analysis system using BERT and LSTM to analyze real-world text across multiple domains with improved accuracy and generalization.

---

## 🚀 Features
- Cross-domain sentiment analysis  
- BERT for contextual understanding  
- LSTM for sequential learning  
- Handles real-world text data  
- Performance evaluation using ML metrics  

---

## 🧠 Technologies Used
- Python  
- TensorFlow  
- Transformers (BERT)  
- NumPy, Pandas  
- Scikit-learn  

---

1. Install dependencies:

multidomain.py
